# absolute + transform

<style>
    .pch-container{
        height: 500px;
        background: skyblue;
        position: relative;
    }
    .pch-item {
        background: bisque;
        width: 50%;
        text-align: center;
        position: absolute;
        left: 50%;
        top: 50%;
        transform: translate(-50%, -50%);
    }
</style>

<div class="pch-container">
    <div class="pch-item">
        <p>水平垂直居中</p>
        <p>水平垂直居中</p>
        <p>水平垂直居中</p>
    </div>
</div>
